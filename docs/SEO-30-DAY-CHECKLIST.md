# Jewel Exchange SEO 30-Day Checklist

Use this as a beginner-friendly execution plan and mark items as you complete them.

## Week 1 — Setup & Indexing

- [ ] Create/verify Google Search Console property for `jewelexchange.lk`
- [ ] Submit sitemap: `https://jewelexchange.lk/sitemap.xml`
- [ ] Confirm indexing for:
  - [ ] `/`
  - [ ] `/jewelry.html`
  - [ ] `/gemstones.html`
  - [ ] `/bespoke.html`
  - [ ] `/about.html`
  - [ ] `/contact.html`
- [ ] Set up or optimize Google Business Profile
- [ ] Ensure NAP is consistent across GBP + website + social profiles

## Week 2 — On-Page SEO

- [ ] Define one primary keyword/theme for each page
- [ ] Refine title tags for CTR + relevance
- [ ] Refine meta descriptions for each page
- [ ] Ensure one clear `H1` per page
- [ ] Improve supporting section headings (`H2/H3`) with clear intent
- [ ] Replace generic image `alt` text with descriptive, search-relevant text

## Week 3 — Technical & Local SEO

- [ ] Run Lighthouse/PageSpeed on top pages and fix highest-impact issues
- [ ] Validate mobile UX across breakpoints (320px, 768px, desktop)
- [ ] Ensure no broken internal links
- [ ] Add/update business photos in Google Business Profile
- [ ] Start collecting customer reviews on Google

## Week 4 — Performance Review & Iteration

- [ ] Review Search Console performance data (queries, CTR, position)
- [ ] Improve pages with high impressions but low CTR
- [ ] Add FAQ-style content to core pages for long-tail queries
- [ ] Publish at least one fresh update (new collection/gemstone highlight)
- [ ] Define a monthly SEO maintenance routine
